title: Setup greenfield project

Set up Patchboard documentation for a new project that has little or no existing source code.

Since this is a greenfield project, you will need human input for most documents. Work through the setup epic (E-0001) tasks in order, asking the user for context at each step.

## Workflow

1. Read the setup epic: `tasks/E-0001/task.md`
2. For each child task in order:
   - Read the task and its plan
   - The plan will direct you to ask the user for context — do so using [How to ask humans to do something](/docs/faq/how-to-ask-humans-to-do-something.md)
   - Write the document using the context provided
   - For how-to guides: verify the documented steps work
   - Deliver each task as a separate PR
3. Update task status as you progress
