---
title: Update PR Documentation
---

Review PR#{{PR_NUMBER}} and update any project documentation affected by the changes. Follow the steps in your role definition.

**Important:** Many PRs do not require documentation changes. If the PR's changes are not reflected in existing documentation (and don't warrant new documentation), report that no updates were needed and move on. Making no changes is a perfectly valid outcome — do not create or update documentation just for the sake of it.
