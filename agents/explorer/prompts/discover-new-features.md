---
title: Discover new features
---

Search for new or undocumented features by comparing the current GUI against the existing function map.

1. Read [Function Map](/docs/product/function-map.md) to understand what is already documented
2. Set up the environment following the **local development** path in your role definition
3. For each function area (route/flow) in the function map, browse to it and look for:
   - New interactive elements not listed in the function map
   - New routes or pages linked from existing pages
   - Changed behaviour of documented elements
   - New modal dialogs, forms, or UI components
4. Do NOT re-document things already in the function map — focus only on what is new or changed
5. Update the function map with your findings, incrementing the version number
6. Deliver via PR with a summary of what was added
